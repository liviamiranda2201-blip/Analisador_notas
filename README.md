# 📊 Analisador de Notas

Este é um projeto simples em Python que analisa notas de alunos.  
Ele permite que o usuário digite várias notas e retorna:

- A **média** das notas  
- Quais notas estão **acima da média**  

---

## 🔹 Funcionalidades

- Recebe notas digitadas pelo usuário  
- Calcula a média da turma  
- Mostra os valores acima da média  

---

## 🔹 Tecnologias usadas

- Python 3  
- Listas  
- Funções  
- Laços (`for`) e condicionais (`if`)  
- Entrada de dados (`input`)
